# :paperclip: Learn NextSSR

This is a project developed on Bootcamp Rocketseat for help me improve my knowledges with NextSSR.

## Installation

Use [yarn](https://classic.yarnpkg.com/pt-BR/docs/install/#debian-stable) or the node package manager [npm](https://www.npmjs.com/get-npm) to install the application.

Run
```bash
yarn
// or npm install
```

## Running

```bash
yarn start
// Or npm start
```
